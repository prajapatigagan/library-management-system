# Library Management System

A simple Library Management System built using Java with Object-Oriented Programming concepts. This application allows basic library operations such as managing books and library services.

## Features
- Add books to the library
- View available books
- Search books
- Manage library records
- Object-Oriented Design
- Simple console-based application

## Technologies Used

### Backend
- Java
- OOP Concepts

### IDE
- IntelliJ IDEA
